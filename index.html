🎯 Fix Mic on Android (Local)
Android Chrome blocks mic for local files. The fix: serve the file via localhost — Chrome treats it as secure. Pick the easiest method below:
📲 HTTP Server App Easiest
  1. 1Install "Simple HTTP Server" from Play Store (free, by Phlox)
    ↗ Open Play Store link
  2. 2Open the app → tap the folder icon → pick the folder with your HTML file
  3. 3Tap Start — note the port (e.g. 8080)
  4. 4Open Chrome and go to:
    http://localhost:8080/rajprep-ai-v14.html
  5. 5Allow mic when Chrome asks → mic works!
💻 Termux (Python server) For Tech Users
  1. 1Install Termux from F-Droid or Play Store
  2. 2Run: pkg install python && cd /sdcard && python -m http.server 8080
  3. 3Open Chrome: http://localhost:8080/rajprep-ai-v14.html
🌐 GitHub Pages (Free HTTPS) Permanent Fix
  1. 1Create a free account at github.com
  2. 2Create a new repo → upload this HTML file
  3. 3Go to Settings → Pages → enable from main branch
  4. 4Access via https://yourname.github.io/repo/file.html — mic always works!
